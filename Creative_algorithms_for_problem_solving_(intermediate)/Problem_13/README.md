minimum sum(S)
====================================

n*n개의 수가 주어진다. (1<=n<=10)

이때 겹치지 않는 각 열과 각 행에서 수를 하나씩 뽑는다.  
(즉, 총 n개의 수를 뽑을 것이다, 그리고 각 수는 100 이하의 값이다.)

이 n개의 수의 합을 구할 때 최솟값을 구하시오.

**입력** 

첫 줄에 n이 입력된다. 다음 줄부터 n+1줄까지 n개씩의 정수가 입력된다.

**출력**  

구한 최소 합을 출력한다.

| 입력 예 | 출력 예     |
|---|---|
| 3 <br> 1 5 3 <br> 2 4 7 <br> 5 3 5| 7 |
